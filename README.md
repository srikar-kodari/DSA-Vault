# DSA

DSA problems, patterns, and notes.

## Structure

- [TUF/](TUF/) - DSA practice.
- [JavaPractice/](JavaPractice/) - Core Java concepts.
- [TUF/Notes/](TUF/Notes/) - Revision notes.

## Topics

`TUF` covers binary search, sliding window, stack/queue, linked list, trees, recursion, heaps, greedy, bit manipulation, strings, POTD, and random problems.

`JavaPractice` covers collections, streams, lambdas, method references, and multithreading.
